FAQ

Common questions

Quick answers about pricing, data, and how Nexo fits your setup.

Is Nexo free to use?

Yes. Nexo is open source under the MIT license. You only pay for the LLM and search APIs you connect — there is no Nexo subscription fee.

Where is my data stored?

Conversations, settings, and workspace data are stored locally in SQLite on your machine. API calls go directly to the providers you configure.

Which LLM providers are supported?

Nexo supports multiple providers through configurable LLM connections. Add your own API keys and models from supported services in Settings.

What is MCP and why does it matter?

Model Context Protocol is an open standard for connecting AI agents to tools and data sources. Nexo treats MCP as a first-class integration path alongside built-in tools.

Can I build from source?

Absolutely. Clone the repository, install Rust and Node dependencies, and follow the development guide in the repo to run Nexo locally or produce your own builds.
